KKLibrary

public enum KKLibrary

The set of available enums for managing the user’s library.

KKLibrary offers the Status enum for managing a show’s status as well as populate a library view. It also offers the SortType enum for managing the way shows are sorted. Which in turn offers the Options enum for managing the sorting order.
